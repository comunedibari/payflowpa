Questo package e' un adattamento del package

org.whole.lang.bindings

contenuto nel mda tool Whole (whole.sourceforge.net).
La versione usata e': 1.0.0.v20070403-1644
Il jar contenente il package (org.whole.lang.bindings) e':
org.whole.lang_1.0.0.v20070403-1644.jar

Il package e' stato adattato, (anziche' integrato), in quanto la versione di
whole usata impiega la jdk 1.5.
Inoltre i requisiti rendono alcune feature "superflue", come le varie politiche 
di scope management. 
Pertanto quella riportata e' una versione semplificata e semanticamente simile 
(non identica) a quella presente in whole.
